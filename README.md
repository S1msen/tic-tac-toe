# Tic Tac Toe

A simple two-player Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

## Features

- Two-player local gameplay (X and O)
- Win and draw detection
- Score tracking across rounds
- Clean, responsive dark-themed UI
